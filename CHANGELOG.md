# Changelog

## [v1.2.0] - 2026-05-15

### 🤖 추천 시스템 개선
- 추천 결과 0개 시 임계값 완화(0.55→0.3) 자동 재시도 폴백 추가
- 추천 필터를 클라이언트 사이드로 전환 (API 재호출 없이 실시간 필터링)
- 필터 다중 선택 지원 (카테고리/팔로워 OR 조건)
- 스타일 키워드 필터 추가 (일상, 감성, 트렌드 등 15개)
- 팔로워 수 필터 범위 변경 (1K-5K / 5K-10K / 10K-30K / 30K-50K / 50K+)
- maxFollowers 파라미터 미적용 버그 수정
- 추천 입력 placeholder 개선 ("브랜드, 상품의 특징, 키워드, 원하는 분위기를 입력하세요")
- 추천 모드 유지 (필터 결과 0개여도 둘러보기로 전환되지 않음)

### 💬 챗봇 개선
- 한글 IME 조합 중 Enter 전송 방지 (composition 이벤트 처리)
- 답변 수신 시 불필요한 자동 스크롤 방지 (사용자 스크롤 위치 유지)
- 질문 유형 분류 개선: 키워드 충돌 시 GPT 폴백 판별 추가
- 상품/브랜드 설명 패턴 자동 인식 (키워드 없이도 추천으로 분류)

### 👤 회원가입/인증
- 회원가입 시 쇼핑몰 이름/URL 저장 누락 수정

### 🎨 UI/UX
- Find Influencers 탭 제거 및 네비게이션 정리
- My 버튼 UI 개선 (UserCircle 아이콘 + 구분선)
- Recommendation 페이지 필터를 버튼 토글 선택형으로 변경
- 추천 전 기본 인플루언서 둘러보기 표시 (전체 대상 + Load More)
- 팔로워 수 직접 구간 입력 필드 추가
- "기타" 카테고리 제거
- 시간 표시 한국 시간(KST) 변환 적용

### 🔧 인프라
- README.md DB 표기 MySQL → PostgreSQL 수정

## [v1.1.0] - 2026-05-13

### 🤖 추천 시스템 개선
- 추천 점수 및 추천 이유(LLM) 표기 기능 추가
- 유사도 임계값 기반 필터링 (관련 없는 결과 제거)
- 쇼핑몰 URL 분석 → 브랜드 분위기 자동 반영
- 추천 시 카테고리/팔로워 필터 지원
- 추천 결과 상세 페이지 (/recommendation/{id})

### 💬 챗봇 리뉴얼
- 자체 채팅 UI 구현 (대화 목록, 새 채팅, 이름 수정, 삭제)
- 대화 히스토리 기반 추천 쿼리 개선
- 추천 결과 전체보기 버튼 연결
- 비로그인 게스트 모드 지원

### 👤 마이페이지
- 프로필 편집 (쇼핑몰 이름/URL)
- 쇼핑몰 분위기 분석 기능
- 추천 기록 리스트 및 상세 조회

### 🎨 UI/UX
- Find Influencers / Recommendation 페이지 분리
- 홈 화면 로그인 후 접근 가능
- 로고/파비콘/타이틀 "LinkD-Match" 브랜딩 적용
- 히어로 타이틀 반응형 처리
- 로그인 시 My 페이지로 자동 이동

### 🔧 인프라
- Chatwoot Rack::Attack 비활성화 (429 에러 해결)
- Docker 디스크 관리 자동화
